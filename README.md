# bigorder-hub
hub for bigorder

## Init your project on local
```
# first time
git submodule update --init --recursive

# update after
git submodule update --recursive --remote
```

## tech stack
+ Golang with Gin
+ Mongodb - mongo-go-driver | mongoose
+ mock.js - random data
